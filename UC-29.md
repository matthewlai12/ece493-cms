# Use Case UC-29: View Conference Schedule

**Goal in Context**:  
Allow an attendee to view the published conference schedule so they can plan which sessions to attend.

**Scope**:  
Conference Management System (CMS)

**Level**:  
User Goal

**Primary Actor**:  
Attendee

**Secondary Actors**:  
CMS Web Interface  
CMS Database

**Trigger**:  
The attendee navigates to the conference schedule page.

---

## Success End Condition
* The attendee successfully views the published conference schedule.

## Failed End Condition
* The attendee cannot view the conference schedule.

---

## Preconditions
* The conference schedule has been finalized and published.
* The CMS schedule viewing functionality is available.
* (Optional) The attendee is authenticated, if schedule access is restricted to registered users.

---

## Main Success Scenario
1. The attendee accesses the CMS.
2. The attendee navigates to the conference schedule section.
3. The CMS retrieves the published conference schedule from the database.
4. The CMS displays the schedule in a readable format (e.g., by date, time, and session).
5. The attendee reviews the schedule to plan session attendance.

---

## Extensions

* **2a**: Attendee is not authenticated and authentication is required  
    * **2a1**: The CMS prompts the attendee to log in before displaying the schedule.

* **3a**: Conference schedule is not available  
    * **3a1**: The CMS displays a message indicating that the schedule has not yet been published.

* **3b**: CMS database is unavailable  
    * **3b1**: The CMS displays an error message indicating that the schedule cannot be retrieved at this time.

* **1a**: CMS web server is unavailable  
    * **1a1**: The CMS displays a system error page indicating temporary service unavailability.

---

## Related Information
* **Priority**: Medium–High  
* **Frequency**: Frequent during the conference period  
* **Open Issues**:  
  * Whether attendees can filter or personalize the schedule  
  * Whether the schedule should be downloadable or printable
