# Use Case UC-02: View Conference Registration Prices

**Goal in Context**:  
Allow a guest user to view the conference registration price list without registering or logging into the system.

**Scope**:  
Conference Management System (CMS)

**Level**:  
User Goal

**Primary Actor**:  
Public User (Guest)

**Secondary Actors**:  
CMS Web Interface  
CMS Database

**Trigger**:  
The guest navigates to the registration or pricing section of the CMS website.

---

## Success End Condition
* The conference registration price list is successfully displayed to the guest.

## Failed End Condition
* The conference registration price list is not displayed and the guest cannot view pricing information.

---

## Preconditions
* The CMS web application is deployed and accessible via a supported web browser.
* The conference registration price list has been defined and stored in the CMS.

---

## Main Success Scenario
1. The guest accesses the CMS homepage.
2. The CMS displays navigation options, including a link to conference registration or pricing information.
3. The guest selects the registration pricing option.
4. The CMS retrieves the conference registration price list from the database.
5. The CMS displays the price list to the guest, showing prices based on attendance type.

---

## Extensions

* **4a**: Registration price list is not available  
    * **4a1**: The CMS displays a message indicating that registration pricing is currently unavailable.

* **4b**: CMS database is unavailable  
    * **4b1**: The CMS displays an error message stating that pricing information cannot be retrieved at this time.

* **1a**: CMS web server is unavailable  
    * **1a1**: The CMS displays a system error page indicating temporary service unavailability.

---

## Related Information
* **Priority**: Medium  
* **Frequency**: Frequent (guests commonly review pricing before deciding to register)  
* **Open Issues**:  
  * Whether prices should be downloadable or printable  
  * Whether discounts (e.g., student or early-bird rates) should be highlighted
